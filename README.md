# InterviewSync

InterviewSync is a real-time collaborative interview platform designed to streamline technical interviews through live coding, video communication, screen sharing, and instant messaging.

## Features

* Real-time code synchronization using WebSockets
* Collaborative coding environment powered by Monaco Editor
* Peer-to-peer video calling using WebRTC
* Screen sharing for technical discussions and demonstrations
* Real-time interview chat
* Room-based interview sessions
* Interview timer and participant management

## Tech Stack

Frontend:

* React.js
* Tailwind CSS
* Monaco Editor

Backend:

* FastAPI
* Python
* MongoDB

Real-Time Communication:

* WebSocket
* WebRTC

## Installation

Backend:

```bash
cd backend
python -m uvicorn main:app --reload
```

Frontend:

```bash
cd frontend
npm install
npm start
```
